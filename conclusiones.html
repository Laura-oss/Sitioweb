¿Para qué puedes usar el código HTML y CSS que trabajaste en este periodo?:

Puedo usar estos códigos para hacer páginas web; el código HTML se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos. El código CSS lo puedo utilizar para arreglar la letras de los parrafos, que tan grande la quiero o el color que quiero que tenga la letra, tambien la puedo usar para arreglar las tablas.
